# Roadmap Milestone 02 Core/SaaS Split

## Objective
Separate OSS base capabilities from premium/private capabilities.

## Delivered Scope
1. `core` owns base domain and standalone operation.
2. `saas` owns premium domain and advanced flows.
3. Technical and API boundaries documented.
4. Submodule strategy established for controlled updates.

## Outcome
Clear dual-track model enabling open-source evolution and private premium extension.
