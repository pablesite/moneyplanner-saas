# Glossary

## Core Terms
1. `core`: Public OSS base product.
2. `saas`: Private premium product stack.
3. `ownership`: Premium ownership model (members/splits/links).
4. `API contract`: Stable endpoint behavior and payload shape.
5. `DoD`: Definition of Done for a milestone/release.

## Quality Terms
1. `lint`: Static style and quality checks.
2. `format`: Auto-format conformance checks.
3. `typecheck`: Static typing validation.
4. `coverage`: Percentage of code executed by tests.
